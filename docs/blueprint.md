# **App Name**: TaskFlow

## Core Features:

- Task Creation: Allow users to create new tasks with titles and descriptions.
- Task Listing: Display tasks in a clear, sortable list.
- Task Editing: Enable users to edit existing tasks to update descriptions.
- Task Deletion: Allow users to delete tasks.
- Task Completion: Allow users to mark tasks as complete.

## Style Guidelines:

- Primary color: Sky blue (#7DD3FC) to convey calmness and focus.
- Background color: Very light blue (#F0F9FF), almost white, for a clean interface.
- Accent color: Lavender (#D8B4FE) to highlight interactive elements.
- Body and headline font: 'PT Sans', a sans-serif font, providing a modern yet warm and inviting user interface. This font is suitable for both headlines and body text.
- Simple and clear icons for task actions (edit, delete, complete).
- Clean and intuitive layout with clear separation of tasks.
- Subtle animations for task completion and reordering.
